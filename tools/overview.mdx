> The small stuff that holds everything else together.

Not every repo is a flagship. These are the utilities, templates, and meta-tools that keep the rest of the portfolio reproducible, observable, and easy to fork.

## Repos in this section

| Repo | What it does |
| --- | --- |
| [mlx-benchmarks](https://github.com/JacobPEvans/mlx-benchmarks) | MLX and local-LLM benchmarking harness. Measure model quality and throughput on Apple Silicon. |
| [orbstack-kubernetes](https://github.com/JacobPEvans/orbstack-kubernetes) | Local Kubernetes cluster on OrbStack with OTEL + Cribl ready to go. Dev-loop K8s without a cloud bill. |
| [secrets-sync](https://github.com/JacobPEvans/secrets-sync) | GitHub Actions secret management automation. Pull from a backend, push to repos, audit drift. |
| [python-template](https://github.com/JacobPEvans/python-template) | Pytest + pre-commit + coverage starter. `gh repo create --template` and go. |
| [unifi-backup-decrypt](https://github.com/JacobPEvans/unifi-backup-decrypt) | Utility to decrypt UniFi controller backups for inspection and migration. |
