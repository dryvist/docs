> Splunk and Cribl architect by day. Building the AI dev pipeline by night. Reef tank in the living room, homelab in the basement, both fully monitored.

## On the clock

Splunk and Cribl consultant specializing in security operations. The day job is architecting SIEM platforms, building detection pipelines, and optimizing data flows for clients with serious ingest volume. The specialty is cutting ingest 30–50% while *improving* security posture — fewer events, but the right ones.

## After hours

Three things happen consistently outside the day job:

- **AI dev pipeline** — Claude, Gemini, Copilot, and local MLX models work as a team. Roadmap and review stay human; everything between gets automated. See [AI development pipeline](/architecture/ai-pipeline).
- **Homelab** — Proxmox cluster, UniFi networking, Home Assistant, Splunk, Cribl. All Terraform-provisioned, Ansible-configured, Nix-orchestrated. See [Homelab](/about/homelab).
- **Reef tank** — 75-gallon saltwater reef with clownfish, corals, and a pistol shrimp. Plus freshwater tanks. The fish have better SLOs than most production systems. See [Reef](/about/reef).

## How this site fits

Every public repo gets a page here that explains *how it fits*. Install and usage stay in each repo's `README.md`. Architecture, philosophy, and cross-repo context live here.

The site is built on Mintlify (free Hobby tier) and lives in [JacobPEvans/docs](https://github.com/JacobPEvans/docs). Everything is MDX in git — if Mintlify ever disappears, the same content runs on Docusaurus, Nextra, or any MDX-aware static generator. No vendor lock-in.
