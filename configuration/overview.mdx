> Terraform builds the box. Ansible makes it useful.

Three Ansible repos cover everything that runs on the provisioned infrastructure: the Proxmox host, the application stack on top, and Splunk Enterprise as a separate concern because of its scale and uptime requirements.

## Role map

| Role | Repo | What it does |
| --- | --- | --- |
| Host config | [Proxmox config](/infrastructure/ansible-proxmox) | ZFS, networking, users, performance tuning, monitoring agents |
| Apps on hosts | [Apps on Proxmox](/infrastructure/ansible-proxmox-apps) | HAProxy, Cribl Edge, Cribl Stream |
| Splunk install | [Splunk install](/observability/ansible-splunk) | Splunk Enterprise — indexers, search heads, license, storage tiering |

All three consume the inventory from [Proxmox provisioning](/infrastructure/terraform-proxmox).
